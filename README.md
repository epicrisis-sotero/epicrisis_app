# 🖥️ Epicrisis Frontend (Vue 3 + Pinia)

Este es el repositorio del cliente web de la Plataforma de Anotación de Epicrisis.

⚠️ **IMPORTANTE:** La documentación centralizada, historias de usuario, arquitectura y guías de este ecosistema han sido migradas al repositorio principal de documentación.

👉 **Por favor, consulta el [Centro de Comando (epicrisis_documentacion)](https://github.com/epicrisis-sotero/epicrisis_documentacion) para entender el proyecto completo.**

## Scripts de Desarrollo

- `npm install`
- `npm run dev` (Inicia servidor Vite en http://localhost:5173)
- `npm run build`
